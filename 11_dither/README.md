## Dithering
Dithering ist eine Technik zur Simulation von Farbtiefen, die ein Ausgabegerät nicht nativ darstellen kann. Statt einen bestimmten Wert direkt auf den nächsten verfügbaren Wert zu runden, wird der Quantisierungsfehler gezielt auf benachbarte Pixel weitergegeben – so stimmt der visuelle Gesamteindruck einer Region, auch wenn jeder einzelne Pixel nur einen von wenigen Werten annehmen kann.

## Algorithmen 
* Floyd-Steinberg, Wirkung: scharf, klassisch
* Atkinson, Wirkung: kontrastreich, grafisch
