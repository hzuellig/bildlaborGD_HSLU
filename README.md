# Algorithmic Image Filters - Bildlabor 2026

Sammlung von p5.js-Experimenten fuer algorithmische Bildmanipulation, Glitch, Pixeloperationen und Shader.

Autorin: Hanna Zuellig  
Kontext: HSLU, Bildlabor 2026

## Demo
– [Live-Preview](https://hzuellig.github.io/bildlaborGD_HSLU/)
– [Data Bending](https://hzuellig.github.io/bildlaborGD_HSLU/databending-01.html)

## Inhalt

- Uebersichtsseite: [index.html](index.html)
- Einzelne Sketches in nummerierten Unterordnern
- Zusatzmaterial in [databending-files](databending-files/)

## Schnellstart

1. Repo klonen.
2. In den Projektordner wechseln.
3. Lokalen Server starten, z. B. in VSCode
4. Im Browser oeffnen:

- http://localhost:8000/
- Oder direkt [index.html](index.html)


## Beispiele

- [00_copy_example](00_copy_example/index.html): Vertikale Bildstreifen mit Zufallsversatz
- [01_imgbrush](01_imgbrush/index.html): Bildpinsel, gespiegelt in mehrere Richtungen
- [02_glitchjpg](02_glitchjpg/index.html): JPG-Byte-Manipulation (Glitch)
- [03_manipulatePixels](03_manipulatePixels/index.html): Direkte Pixeloperationen in Bildbereichen
- [04_PixelPainting](04_PixelPainting/index.html): Farbsortierung und Pixelstrukturen
- [05_ShaderPainting](05_ShaderPainting/index.html): Interaktive Verzerrung per Fragment-Shader
- [06_edgeDetection](06_edgeDetection/index.html): Kantendetektion 
- [10_video_slitscan](10_video_slitscan/index.html): Zeitlich versetzte Videostreifen
- [11_dither](11_dither/index.html): Dithering-Algorithmen
